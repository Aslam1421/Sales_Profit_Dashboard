# 📊 Sales & Profit Dashboard (Microsoft Excel)

## 📌 Project Overview

This project presents an interactive Sales & Profit Dashboard built using Microsoft Excel. The dashboard provides insights into sales performance, profit trends, customer contributions, and regional distribution.

## 📊 Key Metrics

* 💰 Total Sales: $19.2M
* 📈 Total Profit: $247K

## 📌 Features

* Category-wise Sales Analysis (Furniture, Office Supplies, Technology)
* Monthly Sales Trend Analysis
* Profit Analysis by Year
* Top Customers based on Profit
* Sales Distribution by State (Geographical Analysis)
* Interactive Filters (Category, Year, Month slicers)

## 💡 Key Insights

* Technology category generates the highest sales
* Sales peak during November and December
* A few customers contribute significantly to total profit
* Sales distribution highlights key performing states

## 🛠 Tools & Technologies

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning
* Data Visualization
* Dashboard Design
